Software

I am involved in the following FEniCS projects,
  • Dolfin, a problem solving environment for ordinary and partial differential equations,
  • Instant, a Python module for inlining of C/C++ code in Python,
  • SyFi, a package for defining and using finite element methods,
  • UFC, a unified form assembly code format.
  • Finite Element Exterior Calculus
  • Block operators and algebra for DOLFIN
I am also one of the developers of the tools for multilevel methods and mixed elements in Diffpack.
